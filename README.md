# OKX 100x Killer

A behavioral guard Chrome extension that blocks high leverage trading pages on OKX until a full declaration is typed.

## 🚫 Purpose

High leverage trading triggers gambling psychology.

This extension forces the user to type a full declaration before accessing derivatives pages.

It acts as a behavioral circuit breaker.

## 🔒 How It Works

- Detects swap / futures / margin pages
- Locks the page
- Requires full declaration input
- Unlocks once confirmed

## 🧠 Why

Because discipline > strategy.

## ⚙ Installation

1. Download the repo
2. Open chrome://extensions
3. Enable Developer Mode
4. Load unpacked
5. Select the folder

## ⚠ Disclaimer

This tool does not prevent losses.
It only slows down impulsive behavior.
